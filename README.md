# NinjaCart Product Classification 📦🤖

## 📌 Project Overview
This project focuses on building a **Machine Learning classification model** for NinjaCart-related data.  
The goal is to analyze the dataset, preprocess features, and predict product categories using supervised learning techniques.

This project demonstrates skills in:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine learning model building
- Model evaluation


## 🧠 Problem Statement
To classify NinjaCart products into appropriate categories based on the given features, improving automation and decision-making in supply chain management.



## 📊 Dataset
- The dataset contains product-related attributes
- Includes both numerical and categorical features
- Data preprocessing was performed to handle missing values and encoding
